# Kali 工具按使用领域整理

这套文档基于当前目录下的原始工具笔记重新整理，特点是：

- 原始文档保留不动。
- 新增按使用领域划分的目录。
- 每个工具都生成了一个精简速查版，方便比赛时快速翻。

## 分类目录
- [00-总览与环境](00-总览与环境/README/)：整体说明、环境信息、路径与索引。 共 2 篇。
- [01-Web渗透与接口测试](01-Web渗透与接口测试/README/)：面向站点探测、接口调试、目录爆破、SQL 注入与 Web Fuzz。 共 10 篇。
- [02-内网与AD](02-内网与AD/README/)：面向域环境枚举、凭证获取、协议利用与横向移动。 共 6 篇。
- [03-网络扫描与隧道VPN](03-网络扫描与隧道VPN/README/)：面向端口扫描、连通性验证、端口转发、VPN 接入与网络排障。 共 5 篇。
- [04-Pwn与二进制利用](04-Pwn与二进制利用/README/)：面向 ELF 分析、漏洞利用、libc 匹配、ROP 构造与 exploit 开发。 共 13 篇。
- [05-逆向与移动安全](05-逆向与移动安全/README/)：面向 APK 分析、反编译、静态分析、调试与基础二进制观察。 共 8 篇。
- [06-取证与流量分析](06-取证与流量分析/README/)：面向文件恢复、镜像取证、日志排查、流量抓取与样本检测。 共 11 篇。
- [07-密码学与口令攻击](07-密码学与口令攻击/README/)：面向哈希识别、离线破解、在线爆破、TLS/编码与证书处理。 共 6 篇。
- [08-通用脚本与系统运维](08-通用脚本与系统运维/README/)：面向快速写脚本、服务管理、终端交互与日常辅助。 共 2 篇。
- [09-字典与资源库](09-字典与资源库/README/)：面向字典、payload、用户名密码库等通用资源。 共 2 篇。

## 使用建议
- 先看最接近题型的分类 README，再打开具体工具速查页。
- 需要详细参数时，跳回每页顶部的“原始文档”链接。
- Pwn 相关建议搭配 `checksec`、`readelf`、`gdb`、`pwntools`、`ROPgadget` 连着看。
- Web 相关建议先看目录扫描类，再看抓包代理与注入利用类。
- 取证与流量题通常按“文件识别 -> 提取恢复 -> 流量过滤 -> 规则检测”顺序排查。
