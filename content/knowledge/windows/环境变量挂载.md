## 挂载范围

- 挂载位置：Windows 用户级 `Path` 环境变量。
- 生效方式：新打开的终端会自动生效；当前已经打开的终端可能需要重开，或者手动刷新 `$env:Path`。
- 处理原则：只挂载实际存在、加入 PATH 后有明确命令入口的目录。源码目录、uTools 插件、配置文件目录、容易产生命令名冲突的目录没有强行挂载。
- Kali 侧没有新增 PATH：SSH 环境当前 PATH 是 `/usr/local/sbin:/usr/local/bin:/usr/sbin:/usr/bin:/sbin:/bin:/usr/local/games:/usr/games`，已覆盖 Kali 中已安装工具所在的 `/usr/bin`、`/usr/sbin`、`/usr/local/bin`。
