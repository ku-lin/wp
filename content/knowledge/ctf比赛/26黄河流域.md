## 题解精选

### 1. Flow

- 判断：Windows PE 逆向题，外层把 `UPX` 壳伪装成 `VMP`。
- 关键线索：把节名 `VMP0/VMP1/VMP2` 和魔数 `VMP!` 改回 `UPX0/UPX1/UPX2`、`UPX!` 后即可正常解包。
- 解题步骤：
  1. 先修壳并 `upx -d` 解包。
  2. 第一层逆出伪 flag `flag{why_you_think_it_is_flag}`。
  3. 继续逆第二层自定义块加密，恢复真实中间串 `Y0u@regrEatreveRser_1145`。
- 结果判断：最终 flag 为 `flag{Y0u@regrEatreveRser_1145}`。
- 题解来源：`S:\study\比赛\26黄河流域\Flow.md`

### 2. 川味小厨

- 判断：不是单纯业务逻辑题，真正目标是通过后台模板执行读取 `/flag`。
- 关键线索：
  - JWT 使用 `ES256`，但服务端接受 DER 格式的零签名。
  - `role=admin` 会被拦截，但 `ad\u006din` 能绕过过滤器并在业务层还原成 `admin`。
  - 上传接口可把文件写到 `../templates/dashboard.html`。
- 解题步骤：
  1. 伪造零签名 JWT，payload 中把角色写成 `ad\u006din`。
  2. 调 `/api/user/profile` 读取 admin 用户信息。
  3. 用拿到的管理员密码正常登录后台。
  4. 上传恶意 Thymeleaf 模板，访问 `/admin/dashboard` 读取 `/flag`。
- 结果判断：输出 `flag{sanjiu}`。
- 题解来源：`S:\study\比赛\26黄河流域\川味小厨.md`

### 3. λd

- 判断：RSA 变体，弱点不在普通 `phi(N)`，而在私钥定义使用了 `(p^2 - 1)(q^2 - 1)`。
- 关键线索：
  - `p` 和 `q` 的差值较小。
  - 可把目标式化成只含 `N` 和 `x = p - q` 的关系：`e*d - 1 = k * ((N - 1)^2 - x^2)`。
- 解题步骤：
  1. 构造二元小根方程。
  2. 对变量 `(k, x)` 使用 LLL / Coppersmith 求出 `x = p - q`。
  3. 由 `x^2 + 4N` 开平方恢复 `p + q`，完成分解。
  4. 反推出 `d`，再执行 `pow(c, d, N)` 解密。
- 结果判断：最终 flag 为 `flag{Lattice_LLL_Defeats_Large_Delta_RSA_Variants!}`。
- 题解来源：`S:\study\比赛\26黄河流域\λd.md`

### 4. Double²

- 判断：并非普通 RSA 分解，而是基于 `2^k` 次幂剩余构造的按位加密。
- 关键线索：
  - 公开参数 `g` 可以推出 `sqrt(-1) mod N`。
  - 在高斯整数 `Z[i]` 上计算四次 Jacobi 符号后，每一位样本会自然分成两类。
- 解题步骤：
  1. 先在 `Z[i]` 中求 `gcd_Z[i](N, g^64 + i)`，得到 `N = a^2 + b^2` 的表示。
  2. 对每一行样本计算 quartic Jacobi symbol。
  3. 如果一整行结果全为 `1`，判作 bit `1`；否则判作 bit `0`。
  4. 最后按字节拼回明文。
- 结果判断：最终 flag 为 `flag{J1st_P01y_RSA_Can_s01ve_1t}`。
- 题解来源：`S:\study\比赛\26黄河流域\Double².md`

### 5. cake

- 判断：Misc 题，核心是 ZipCrypto 内部 key + 视频像素隐写。
- 关键线索：
  - `cake_knife.txt` 中的 3 个十六进制数正好对应 ZipCrypto 三段内部状态。
  - 用这三段 key 解开 `cake_base.bin` 后，文件头变成 `PK\x03\x04`。
  - 真正 flag 藏在 `scream.avi` 固定像素点 `(123, 45)` 的 RGB 通道中。
- 解题步骤：
  1. 根据 ZipCrypto keystream 公式解密 `cake_base.bin`。
  2. 解压得到 `fruit.bin`、`scream.avi`、`instruction.txt`。
  3. 逐帧读取 `scream.avi` 指定像素，提取可打印 ASCII。
- 结果判断：输出 `flag{W0w_d3lici0us_c4ke!!}`。
- 题解来源：`S:\study\比赛\26黄河流域\cake.md`
